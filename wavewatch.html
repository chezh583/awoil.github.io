Wavewatch III

Download Wavewatch III (WW3) source code:

git clone https://github.com/NOAA-EMC/WW3

This will create a folder named as 'WW3'. Go to the 'WW3' directory, and start setting up:

./model/bin/w3_setup model

This will compile auxiliary programs and setup the environment file wwatch3.env

Gridgen

Gridgen is a MATLAB program to create files that contain bathimethric information to be used in WW3. Download Gridgen:

git clone https://github.com/NOAA-EMC/gridgen

Run the 'setup_gridgen.m' function, automatically downloading bathimethric data:

matlab setup_gridgen.m

OASIS-MCT

In order to couple WW3 with other climate models (ice, ocean or atmosphere model), OASIS-MCT needs to be installed:

git clone https://gitlab.com/cerfacs/oasis3-mct.git
