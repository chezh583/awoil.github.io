Cheat Sheet for Unix Commands

You will get some useful unix commands here.

Command Function
cd dir go to a directory
cd .. go to previous directory
cp [] dir/fname [] newdir copy a file to another directory
ls []. Options: -a, -l list.
pwd current directory
rm fname; rm -r dir remove a file/directory
vim fname create and edit a file
mv fname1 fname2 rename fname1 to fname2
mkdir dir create a directory
rmdir remove a directory
mv fname dir move a file to a directory
cp [] dir/fname [] newdir copy a file to another directory
df -h check storage
chmod [] fname change access permissions, further read View
source appname execute an application

When viewing or editing a file (e.g. using vi command), use following command.

Command Function
i insert, able to edit
:wq save and quit
q quit
q! quit without save

Other useful commands.

Command Function
* all. e.g. "*.dat" means all files with ".dat" as names
module avail modulename check available module
modulename --version check the version of module
modulmatlab -nodisplay < fname run matlab code without opening application
zip fname.zip fname1 fname2; zip dir.zip dir1 dir2 zip files or directories
unzip fname.zip unzip a zip file
tar -zcvf fname.tgz fname1 fname2; tar -zcvf fname.tgz dir1 dir2 zip files or directories
tar -zxvf fname.tgz unzip a zip file
